# GestionDeVentanasJavaFX_3

Proyecto desarrollado en Java utilizando JavaFX y Maven para la gestión de ventanas y navegación entre escenas en aplicaciones de escritorio.

## Descripción

Esta aplicación continúa con las prácticas de JavaFX enfocadas en la gestión de múltiples ventanas, permitiendo la navegación entre diferentes vistas y el control de escenas dentro de la aplicación.

El proyecto sirve como práctica para consolidar el manejo de estructuras de navegación en aplicaciones JavaFX.

## Tecnologías utilizadas

- Java
- JavaFX
- Maven

## Requisitos

- JDK 17 o superior
- Maven instalado
- IDE compatible con JavaFX (IntelliJ IDEA, Eclipse o NetBeans)
